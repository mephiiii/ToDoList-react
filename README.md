# React Todo List

A fully responsive Todo List web application built with React.

## Features
- Add new tasks
- Delete tasks
- Complete tasks
- Task priority (High / Medium / Low)
- Dark / Light theme
- Form validation with Zod
- API data management with MockAPI

## Technologies
- React
- React Router
- React Query
- Tailwind CSS
- Zod
- MockAPI

## Pages
- `/` : Active tasks
- `/complete` : Completed tasks
